# Introduction to Discrete Numeric Functions (Section 9.1)

---

A discrete numeric function is the mathematical foundation of sequences. In computer science, we often work with quantities that change at discrete steps (e.g., execution time of an algorithm for inputs of size $n$, memory usage, or clock cycles). These are represented as discrete numeric functions.

---

### 1. Definition and Notation

> **Definition 1:** A **discrete numeric function** (or simply **numeric function**) is a real-valued function whose domain is the set of non-negative integers $\mathbb{N}_0 = \{0, 1, 2, \dots\}$.

We typically denote a numeric function as $a$, $b$, $c$, etc. The value of the function $a$ at a non-negative integer $n$ is denoted as $a(n)$ or $a_n$.
* The function can be represented as a sequence: $a_0, a_1, a_2, \dots, a_n, \dots$
* The domain is always $\{0, 1, 2, \dots\}$, and the codomain is $\mathbb{R}$.

---

### 2. Standard Examples of Numeric Functions

Below are common mathematical forms of numeric functions:

1. **Constant Function:**
   $$a(n) = c \quad (\text{for a constant } c)$$
   Sequence: $c, c, c, c, \dots$

2. **Geometric Function:**
   $$a(n) = \alpha^n \quad (\text{for a constant } \alpha)$$
   Sequence: $1, \alpha, \alpha^2, \alpha^3, \dots$

3. **Polynomial Function:**
   $$a(n) = n^k \quad (\text{for an integer } k \ge 1)$$
   For $k=1$, sequence: $0, 1, 2, 3, 4, \dots$
   For $k=2$, sequence: $0, 1, 4, 9, 16, \dots$

4. **Piecewise Function:**
   $$a(n) = \begin{cases} 2^n & \text{if } 0 \le n \le 3 \\ 0 & \text{if } n \ge 4 \end{cases}$$
   Sequence: $1, 2, 4, 8, 0, 0, 0, \dots$

---

### 3. Visualizing Numeric Functions

We can visualize a numeric function $a(n)$ as a sequence of discrete points on a Cartesian plane where the x-axis represents $n$ and the y-axis represents $a(n)$. Unlike continuous functions, we do not draw a continuous line between these points.

```
 a(n)
  ^
  |        o [a(3) = 9]
  |
  |
  |
  |     o [a(2) = 4]
  |
  |  o [a(1) = 1]
  |o [a(0) = 0]
--+--+--+--+--------------> n
  0  1  2  3
```

### 🧠 Check for Understanding

1. Write the first 5 terms (indices $0, 1, 2, 3, 4$) of the numeric function:
   $$a(n) = 3^n - 2^n$$

2. True or False: The domain of a discrete numeric function can contain negative integers.

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step solutions:
> 
> 1. **First 5 terms of $a(n) = 3^n - 2^n$:**
>    * $a(0) = 3^0 - 2^0 = 1 - 1 = 0$
>    * $a(1) = 3^1 - 2^1 = 3 - 2 = 1$
>    * $a(2) = 3^2 - 2^2 = 9 - 4 = 5$
>    * $a(3) = 3^3 - 2^3 = 27 - 8 = 19$
>    * $a(4) = 3^4 - 2^4 = 81 - 16 = 65$
>    * The sequence is: $0, 1, 5, 19, 65$.
> 
> 2. **Domain of Numeric Functions:**
>    * **False.** By definition, the domain of a discrete numeric function is strictly the set of non-negative integers $\mathbb{N}_0 = \{0, 1, 2, \dots\}$.
