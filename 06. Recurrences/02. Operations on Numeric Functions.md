# Operations on Numeric Functions (Section 9.2)

---

Just as we manipulate real-valued functions in continuous calculus, we can define several algebraic and shifting operations on discrete numeric functions. These operations allow us to model discrete systems and transitions.

---

### 1. Basic Algebraic Operations

Let $a$ and $b$ be two discrete numeric functions, and let $\alpha$ be a real scalar.

* **Addition ($a + b$):** The sum $c = a + b$ is defined pointwise for all $n \ge 0$:
  $$c(n) = a(n) + b(n)$$

* **Pointwise Product ($ab$):** The product $d = ab$ is defined pointwise for all $n \ge 0$:
  $$d(n) = a(n) \cdot b(n)$$

* **Scalar Multiplication ($\alpha a$):** The scalar multiple $e = \alpha a$ is defined pointwise for all $n \ge 0$:
  $$e(n) = \alpha a(n)$$

---

### 2. Shift Operators

Shift operators allow us to delay or advance the sequence values in time.

#### A. Forward Shift (Delay) Operator ($S^k$):
For a positive integer $k$, the forward shift operator $S^k$ shifts the sequence to the right by $k$ positions, padding the first $k$ positions with $0$.
$$S^k a(n) = \begin{cases} 0 & \text{if } 0 \le n < k \\ a(n - k) & \text{if } n \ge k \end{cases}$$

**Example:** If $a = (1, 2, 4, 8, \dots)$, then:
* $S^1 a = (0, 1, 2, 4, 8, \dots)$
* $S^2 a = (0, 0, 1, 2, 4, \dots)$

#### B. Backward Shift (Advance) Operator ($S^{-k}$):
For a positive integer $k$, the backward shift operator $S^{-k}$ shifts the sequence to the left by $k$ positions, discarding the first $k$ terms.
$$S^{-k} a(n) = a(n + k) \quad (\text{for all } n \ge 0)$$

**Example:** If $a = (1, 2, 4, 8, 16, \dots)$, then:
* $S^{-1} a = (2, 4, 8, 16, \dots)$
* $S^{-2} a = (4, 8, 16, \dots)$

---

### 3. Difference and Accumulation Operators

* **Forward Difference ($\Delta a$):** Measures the change between consecutive steps:
  $$\Delta a(n) = a(n + 1) - a(n) = S^{-1} a(n) - a(n)$$

* **Backward Difference ($\nabla a$):**
  $$\nabla a(n) = \begin{cases} a(0) & \text{if } n = 0 \\ a(n) - a(n - 1) & \text{if } n \ge 1 \end{cases}$$

* **Accumulated Sum:** Defines a new function $b(n)$ representing the running sum of values up to index $n$:
  $$b(n) = \sum_{i=0}^{n} a(i)$$

---

### 4. Convolution ($a * b$)

The convolution of two numeric functions is a fundamental operation in discrete signal processing and recurrences.

> **Definition 1:** The **convolution** of two numeric functions $a$ and $b$, denoted by $c = a * b$, is defined as:
> $$c(n) = \sum_{i=0}^{n} a(i) b(n - i)$$

* **Commutativity:** Convolution is commutative, meaning $a * b = b * a$.
  $$c(n) = \sum_{i=0}^{n} a(i) b(n - i) = \sum_{i=0}^{n} b(i) a(n - i)$$

#### 📝 Detailed Worked Example:
Find the convolution of $a(n) = 2^n$ and $b(n) = 3^n$.

1. Using the definition of convolution:
   $$c(n) = \sum_{i=0}^{n} a(i) b(n - i) = \sum_{i=0}^{n} 2^i \cdot 3^{n - i}$$
2. Factor out $3^n$ from the sum:
   $$c(n) = 3^n \sum_{i=0}^{n} 2^i \cdot 3^{-i} = 3^n \sum_{i=0}^{n} \left(\frac{2}{3}\right)^i$$
3. Apply the geometric series summation formula $\sum_{i=0}^{n} x^i = \frac{1 - x^{n+1}}{1 - x}$ with $x = 2/3$:
   $$\sum_{i=0}^{n} \left(\frac{2}{3}\right)^i = \frac{1 - (2/3)^{n+1}}{1 - 2/3} = \frac{1 - (2/3)^{n+1}}{1/3} = 3 \left[1 - \left(\frac{2}{3}\right)^{n+1}\right]$$
4. Multiply by $3^n$:
   $$c(n) = 3^n \cdot 3 \left[1 - \left(\frac{2}{3}\right)^{n+1}\right] = 3^{n+1} \left[1 - \left(\frac{2}{3}\right)^{n+1}\right]$$
5. Distribute $3^{n+1}$:
   $$c(n) = 3^{n+1} - 3^{n+1} \cdot \frac{2^{n+1}}{3^{n+1}} = 3^{n+1} - 2^{n+1}$$
6. **Solution:** The convolution is $c(n) = 3^{n+1} - 2^{n+1}$.

### 🧠 Check for Understanding

1. Let $a(n) = n$ for $n \ge 0$. Find the sequence values of $S^2 a$ for $n = 0, 1, 2, 3, 4$.
2. Compute the value of $(a * b)(2)$ where $a(n) = n + 1$ and $b(n) = 2^n$.

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step solutions:
> 
> 1. **Forward Shift $S^2 a(n)$:**
>    * The original sequence is $a = (0, 1, 2, 3, 4, 5, \dots)$.
>    * Shifting right by 2 and padding with $0$s gives:
>      * For $n=0$: $S^2 a(0) = 0$
>      * For $n=1$: $S^2 a(1) = 0$
>      * For $n=2$: $S^2 a(2) = a(0) = 0$
>      * For $n=3$: $S^2 a(3) = a(1) = 1$
>      * For $n=4$: $S^2 a(4) = a(2) = 2$
>    * The sequence is: $0, 0, 0, 1, 2$.
> 
> 2. **Convolution $(a * b)(2)$:**
>    * Pointwise values:
>      * $a(0) = 1, \quad a(1) = 2, \quad a(2) = 3$
>      * $b(0) = 1, \quad b(1) = 2, \quad b(2) = 4$
>    * Apply convolution definition for $n = 2$:
>      $$c(2) = a(0)b(2) + a(1)b(1) + a(2)b(0)$$
>      $$c(2) = (1 \cdot 4) + (2 \cdot 2) + (3 \cdot 1) = 4 + 4 + 3 = 11$$
