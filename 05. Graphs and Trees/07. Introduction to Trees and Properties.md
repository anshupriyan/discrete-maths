# Introduction to Trees and Properties (Section 11.1)

---

Trees are a particularly important subclass of graphs. In computer science, trees are used to model hierarchical structures (like file directories, HTML DOM trees, or organizational charts), represent expressions, and optimize searches.

---

### 1. Definitions: Trees and Forests

> **Definition 1:** A **tree** is a connected undirected graph with no simple circuits.
> 
> **Definition 2:** A **forest** is an undirected graph with no simple circuits. Each connected component of a forest is a tree.

#### Key Characterization:
Because a tree is connected and has no simple circuits, it represents the absolute minimum set of edges required to connect a set of vertices.

> **Theorem 1 (Unique Path Theorem):** An undirected graph is a tree if and only if there is a unique simple path between any two of its vertices.

#### Proof Sketch:
1. **If $G$ is a tree:** Suppose there are two different simple paths between $u$ and $v$. By combining parts of these paths, we could form a simple circuit, contradicting the definition of a tree. Thus, the path must be unique.
2. **If there is a unique simple path between any two vertices:** Since there is a path between any two vertices, the graph is connected. Since the path is unique, there can be no simple circuits (as a simple circuit would imply two different simple paths between any of its vertices). Thus, the graph is a tree. $\blacksquare$

---

### 2. The Tree Edge Theorem

A fundamental property of trees is the exact mathematical relationship between the number of vertices and the number of edges.

> **Theorem 2 (Tree Edge Theorem):** A tree with $n$ vertices has exactly $n - 1$ edges.

#### Proof (By Mathematical Induction):
We will prove this theorem using induction on the number of vertices $n$.

1. **Base Case ($n = 1$):** A tree with $1$ vertex has $0$ edges. Since $n - 1 = 1 - 1 = 0$, the theorem holds.
2. **Inductive Hypothesis:** Assume the theorem is true for all trees with $k$ vertices, where $k \ge 1$. That is, any tree with $k$ vertices has exactly $k-1$ edges.
3. **Inductive Step:** We want to show that any tree $T = (V, E)$ with $k + 1$ vertices has exactly $k$ edges.
   * Let $T$ be a tree with $k + 1$ vertices. Since $T$ is connected and has $k+1 \ge 2$ vertices, it must have at least one edge $e = \{u, v\}$.
   * If we remove the edge $e$ from $T$ to form the subgraph $T - e$, we disconnect the graph. Specifically, because there was a unique simple path between $u$ and $v$ in $T$ (Theorem 1), removing $e$ leaves no path between $u$ and $v$.
   * Thus, $T - e$ consists of exactly two connected components, say $T_1$ and $T_2$.
   * Since $T$ contains no simple circuits, neither $T_1$ nor $T_2$ can contain simple circuits. Since they are also connected, both $T_1$ and $T_2$ are trees.
   * Let $n_1$ and $n_2$ be the number of vertices in $T_1$ and $T_2$ respectively. Because the vertices of $T$ are partitioned between the two components:
     $$n_1 + n_2 = k + 1$$
   * Since $n_1 \ge 1$ and $n_2 \ge 1$, we must have $n_1 \le k$ and $n_2 \le k$.
   * We can therefore apply the inductive hypothesis to both $T_1$ and $T_2$:
     * $T_1$ has $n_1 - 1$ edges.
     * $T_2$ has $n_2 - 1$ edges.
   * The total number of edges in $T$ is the sum of the edges in $T_1$ and $T_2$, plus the single edge $e$ that we removed:
     $$|E| = (n_1 - 1) + (n_2 - 1) + 1 = n_1 + n_2 - 1$$
   * Substituting $n_1 + n_2 = k + 1$:
     $$|E| = (k + 1) - 1 = k$$
   * This proves that a tree with $k+1$ vertices has exactly $k$ edges.
   * By mathematical induction, the theorem is true for all positive integers $n \ge 1$. $\blacksquare$

### 🧠 Check for Understanding

1. A forest consists of 5 trees and has a total of 50 vertices. How many edges does the forest have?
2. Can a simple graph with 6 vertices and 6 edges be a tree?

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step solutions:
> 
> 1. **Forest Edges Calculation:**
>    * Let the forest have $k$ trees: $T_1, T_2, \dots, T_k$ where $k = 5$.
>    * Let $n_i$ be the number of vertices in tree $T_i$, and $e_i$ be the number of edges in tree $T_i$.
>    * By Theorem 2, each tree $T_i$ has $e_i = n_i - 1$ edges.
>    * The total number of edges in the forest is:
>      $$\text{Total Edges} = \sum_{i=1}^5 e_i = \sum_{i=1}^5 (n_i - 1) = \left(\sum_{i=1}^5 n_i\right) - 5$$
>    * Since the total number of vertices is $\sum_{i=1}^5 n_i = 50$:
>      $$\text{Total Edges} = 50 - 5 = 45 \text{ edges}$$
> 
> 2. **Verification of Tree Properties:**
>    * **No.** By Theorem 2, any tree with $V = 6$ vertices must have exactly $E = V - 1 = 5$ edges. Since this graph has 6 edges, it violates the tree edge property. (Specifically, a connected graph with $V$ vertices and $V$ edges must contain at least one simple circuit).
