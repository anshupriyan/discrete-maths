# Partial Functions (Section 2.3.6)

---

### 1. The Core Motivation: Why Do We Need Partial Functions?

In the standard definition of a function, an output **must** be assigned to **every single element** in the domain. However, real-world computer science and abstract math scenarios frequently feature cases where this requirement fails:

* **In Computer Science:** A program designed to compute a function might crash, hit an **infinite loop**, a **division-by-zero error**, or an **integer overflow** for certain inputs, failing to return a result.
* **In Abstract Mathematics:** Formulas are often undefined for specific real numbers. For example, $f(x) = \frac{1}{x}$ is undefined at $x=0$, $f(x) = \sqrt{x}$ is undefined for negative real numbers, and $f(x) = \arcsin(x)$ is only defined on the interval $[-1, 1]$.
* **In Everyday Contexts:** A "youngest child" assignment function is completely undefined for a couple that has no children.

To model these situations rigorously, discrete mathematics introduces the concept of a **partial function**.

---

### 2. Formal Mathematical Definition

The textbook defines a partial function and introduces key vocabulary to describe its structure:

> **Definition 13:** A **partial function** $f$ from a set $A$ to a set $B$ is an assignment to each element $a$ in a *subset* of $A$, called the **domain of definition** of $f$, of a unique element $b$ in $B$.
> * The set $A$ is still called the **domain** of $f$, and $B$ is called the **codomain** of $f$.
> * We say that $f$ is **undefined** for elements in $A$ that are not in the domain of definition of $f$.
> * When the domain of definition exactly equals $A$, we say $f$ is a **total function** (which is just a standard, normal function).

* **Notation Note:** We still write $f: A \rightarrow B$ for partial functions. The surrounding context determines whether an author implies a partial or total function.

#### **Textbook Example 34:**
The function $f : \mathbf{Z} \rightarrow \mathbf{R}$ where $f(n) = \sqrt{n}$ is a partial function from the integers ($\mathbf{Z}$) to the real numbers ($\mathbf{R}$). Identify its core components.

* **Domain:** $\mathbf{Z}$ (All integers).
* **Codomain:** $\mathbf{R}$ (All real numbers).
* **Domain of Definition:** The set of all **nonnegative integers** ($\{0, 1, 2, 3, \dots\}$).
* **Where it is Undefined:** Undefined for all **negative integers** ($\{\dots, -3, -2, -1\}$) because negative numbers do not have real square roots.
* **Is it Total?** No, because the domain of definition does not include all integers in $\mathbf{Z}$.

---

### 3. Representing "Undefined" in Programming

In computer science, we often need to represent partial functions without causing programs to crash. 

To achieve this, we can add a special abstract token element—often denoted as **$u$** (standing for *undefined*, or represented as `NaN`, `null`, `None`, or `nil` in programming)—to the codomain.

As noted in **Exercise 80**, a partial function $f: A \rightarrow B$ can be transformed into a standard, fully functional **total function** $f^*: A \rightarrow B \cup \{u\}$ by using the following definition rule:

$$f^*(a) = \begin{cases} f(a) & \text{if } a \text{ belongs to the domain of definition of } f \\ u & \text{if } f \text{ is undefined at } a \end{cases}$$

### 🧠 Quick Check: Try it Yourself!

Consider the partial function $f: \mathbf{Z} \rightarrow \mathbf{R}$ defined by $f(n) = \frac{1}{n}$.

1. What is its **domain** and **codomain**?
2. What is its **domain of definition**?
3. For what specific integer value(s) is it **undefined**?
4. Is it a **total function**?

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **Domain and Codomain:**
>    * **Domain:** $\mathbf{Z}$ (the set of all integers).
>    * **Codomain:** $\mathbf{R}$ (the set of all real numbers).
> 2. **Domain of Definition:** $\mathbf{Z} - \{0\}$ (all non-zero integers). 
> 3. **Where it is Undefined:** $n = 0$. Division by zero is mathematically undefined.
> 4. **Is it a Total Function?** **No**. The domain of definition ($\mathbf{Z} - \{0\}$) does not equal the full domain ($\mathbf{Z}$), which means it is a partial function.
