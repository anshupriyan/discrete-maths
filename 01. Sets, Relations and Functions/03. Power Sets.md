# Power Sets (Section 2.1.5)

---

### 1. What is a Power Set?

When working with sets, we frequently need to look at all the possible ways we can group their elements into smaller collections. This leads directly to the definition of a power set.

> **Definition 6:** Given a set $S$, the **power set** of $S$ is the set of all subsets of $S$. The power set of $S$ is denoted by $\mathcal{P}(S)$.

**Crucial Structural Rule:** Every single element inside a power set is itself a *set*.

---

### 2. Textbook Examples

The textbook provides three fundamental examples showing how this definition operates under different conditions:

#### Example 14: A Standard Finite Set
What is the power set of the set $\{0, 1, 2\}$?

* **Solution:** To find $\mathcal{P}(\{0, 1, 2\})$, we must list every single possible subset of $\{0, 1, 2\}$.
  1. The empty set (which has 0 elements): $\emptyset$
  2. All subsets with exactly 1 element: $\{0\}, \{1\}, \{2\}$
  3. All subsets with exactly 2 elements: $\{0, 1\}, \{0, 2\}, \{1, 2\}$
  4. The original set itself (with 3 elements): $\{0, 1, 2\}$
* Putting these all together as elements inside a new outer set gives us the complete power set:
  $$\mathcal{P}(\{0, 1, 2\}) = \big\{ \emptyset, \{0\}, \{1\}, \{2\}, \{0, 1\}, \{0, 2\}, \{1, 2\}, \{0, 1, 2\} \big\}$$

#### Example 15: The Power Set of the Empty Set
What is the power set of the empty set, $\mathcal{P}(\emptyset)$?

* **Solution:** The empty set has exactly one subset, which is itself ($\emptyset$).
* Therefore, its power set is a singleton set containing just that one element:
  $$\mathcal{P}(\emptyset) = \{\emptyset\}$$

#### Example 16: The Power Set of a Power Set
What is the power set of $\{\emptyset\}$? *(Remember from Section 2.1.1 that $\{\emptyset\}$ is a set containing one element).*

* **Solution:** Since the set $\{\emptyset\}$ has one element (the empty set), its subsets are:
  1. The empty set: $\emptyset$
  2. The set containing its element: $\{\emptyset\}$
* Gathering these into the power set gives:
  $$\mathcal{P}(\{\emptyset\}) = \{\emptyset, \{\emptyset\}\}$$

---

### 3. The Cardinality Rule for Power Sets

The textbook highlights a crucial mathematical theorem regarding the size of a power set:

If a set has $n$ elements, then its power set has exactly **$2^n$** elements.

We can verify this using the exact examples from the text:
* For the empty set $\emptyset$, the size $n = 0$. Its power set $\mathcal{P}(\emptyset)$ has $2^0 = 1$ element.
* For the set $\{\emptyset\}$, the size $n = 1$. Its power set $\mathcal{P}(\{\emptyset\})$ has $2^1 = 2$ elements.
* For the set $\{0, 1, 2\}$, the size $n = 3$. Its power set $\mathcal{P}(\{0, 1, 2\})$ has $2^3 = 8$ elements.

---

### 🧠 Quick Study Reflection

Based on this explicit section, if a set $A$ has a cardinality of 4 (meaning $|A| = 4$):
1. What is the cardinality of its power set, $|\mathcal{P}(A)|$?
2. Out of all those elements in the power set, how many of them are **proper subsets** of $A$?

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **Cardinality of the Power Set ($|\mathcal{P}(A)|$):** 16. Using the $2^n$ rule, since $n = |A| = 4$, the size of the power set is $2^4 = 16$ elements.
> 2. **Number of Proper Subsets:** 15. A subset is proper if it is not equal to the original set. Since only one subset of $A$ is equal to $A$ itself (the set $A$), the number of proper subsets is $2^n - 1$. For $n = 4$, $16 - 1 = 15$.
