# Introduction to Functions (Sections 2.3 - 2.3.1)

---

### 1. The Core Concept of a Function

In everyday applications, we frequently need to assign each element of one collection to a specific element of another collection.

* **Everyday Example:** In a class where each student is assigned a final letter grade from the set $\{A, B, C, D, F\}$:
  * Adams $\rightarrow A$
  * Chou $\rightarrow C$
  * Goodfriend $\rightarrow B$
  * Rodriguez $\rightarrow A$
  * Stevens $\rightarrow F$

This assignment of every member of a first group to *exactly one* member of a second group is called a **function**.

#### **Why Functions Matter**
Functions are foundational throughout computer science and math. In discrete mathematics, they are explicitly used to define structures like **sequences** and **strings**. They are also used to analyze the **computational complexity** of algorithms (how long a program takes to solve a problem of a given size) and to write subroutines that compute specific output values.

---

### 2. Formal Mathematical Definition

The textbook formalizes this concept in **Definition 1**:

> **Definition 1:** Let $A$ and $B$ be nonempty sets. A **function** $f$ from $A$ to $B$ is an assignment of **exactly one** element of $B$ to each element of $A$.
> 
> We write $f(a) = b$ if $b$ is the unique element of $B$ assigned by the function $f$ to the element $a$ of $A$. If $f$ is a function from $A$ to $B$, we write **$f : A \rightarrow B$**.

* **Alternative Names:** Functions are also referred to as **mappings** or **transformations**.
* **Connection to Relations:** A function $f: A \rightarrow B$ can be defined as a special type of relation from $A$ to $B$ (recall a relation is a subset of the Cartesian product $A \times B$). If a relation contains **one, and only one**, ordered pair $(a, b)$ for every element $a \in A$, it uniquely defines a function where $f(a) = b$.

---

### 3. Domain, Codomain, and Range

When dealing with a function, we must carefully separate the sets involved.

> **Definition 2:** If $f$ is a function from $A$ to $B$:
> * **$A$** is called the **domain** of $f$.
> * **$B$** is called the **codomain** of $f$.
> * If $f(a) = b$, then $b$ is called the **image** of $a$, and $a$ is called a **preimage** of $b$.
> * The **range** (or image) of $f$ is the set of all actual images of elements of $A$.

#### **Codomain vs. Range**
Do not confuse the codomain and the range:
* The **codomain** is the set of *all possible values* the function is allowed to potentially target.
* The **range** is the set of *actual values* that the function successfully hits.
* The range is **always a subset** of the codomain ($\text{Range} \subseteq \text{Codomain}$).

To define a function, you must specify its domain, its codomain, and the mapping rule. Two functions are **equal** if and only if they share the exact same domain, the exact same codomain, and map every element to the exact same value.

---

### 4. Textbook Examples

#### **Textbook Example 1 (The Grades Function):**
What are the domain, codomain, and range of the grading function $G$ described in the introduction?
* **Domain:** The set of students: $\{\text{Adams, Chou, Goodfriend, Rodriguez, Stevens}\}$.
* **Codomain:** The set of possible grades: $\{A, B, C, D, F\}$.
* **Range:** The letters actually used: **$\{A, B, C, F\}$** (no one received a $D$).

#### **Textbook Example 2 (The Age Relation Function):**
Let $R$ be a relation containing the following ordered pairs of students and their ages: $\{(\text{Abdul}, 22), (\text{Brenda}, 24), (\text{Carla}, 21), (\text{Desire}, 22), (\text{Eddie}, 24), (\text{Felicia}, 22)\}$. Specify this function $f$.
* **Mapping:** $f(\text{Abdul}) = 22$, $f(\text{Brenda}) = 24$, etc.
* **Domain:** The set of names: $\{\text{Abdul, Brenda, Carla, Desire, Eddie, Felicia}\}$.
* **Codomain:** A reasonable candidate set containing these age values, such as **positive integers less than 100**.
* **Range:** The exact set of ages that appear: **$\{21, 22, 24\}$**.

#### **Textbook Example 3 (Bit String Truncation):**
Let $f$ be the function that assigns the last two bits of a bit string of length $\ge 2$ to that string. What are its components?
* **Domain:** The set of all bit strings of length 2 or greater.
* **Codomain & Range:** Both are the complete set of all possible 2-bit combinations: **$\{00, 01, 10, 11\}$**.

#### **Textbook Example 4 (Squaring Integers):**
Let $f : \mathbf{Z} \rightarrow \mathbf{Z}$ assign the square of an integer to that integer ($f(x) = x^2$). What are its components?
* **Domain:** $\mathbf{Z}$ (The set of all integers).
* **Codomain:** $\mathbf{Z}$ (The set of all integers).
* **Range:** Only the perfect squares produced by squaring integers: **$\{0, 1, 4, 9, \dots\}$**.

### 5. Adding and Multiplying Functions

If two functions share the exact same domain, and map directly to real numbers ($\mathbf{R}$), we can perform arithmetic on the functions themselves.

> **Definition 3:** Let $f_1$ and $f_2$ be functions from $A$ to $\mathbf{R}$. Then $f_1 + f_2$ and $f_1f_2$ are also functions from $A$ to $\mathbf{R}$, explicitly evaluated at any element $x \in A$ by:
> $$(f_1 + f_2)(x) = f_1(x) + f_2(x)$$
> $$(f_1f_2)(x) = f_1(x) \cdot f_2(x)$$

#### **Textbook Example 6:**
Let $f_1$ and $f_2$ be functions from $\mathbf{R}$ to $\mathbf{R}$ defined by $f_1(x) = x^2$ and $f_2(x) = x - x^2$. Find $f_1 + f_2$ and $f_1f_2$.
* **Sum:** $(f_1 + f_2)(x) = x^2 + (x - x^2) = \mathbf{x}$
* **Product:** $(f_1f_2)(x) = x^2(x - x^2) = \mathbf{x}^3 - \mathbf{x}^4$

---

### 6. The Image of a Subset

We can evaluate a function on an entire sub-collection of elements at once.

> **Definition 4:** Let $f$ be a function from $A$ to $B$, and let $S$ be a subset of $A$ ($S \subseteq A$). The **image of $S$** under the function $f$ is the subset of $B$ that gathers the images of all individual elements in $S$. We write this as:
> $$f(S) = \{t \mid \exists s \in S \, (t = f(s))\} = \{f(s) \mid s \in S\}$$

#### **Textbook Example 7:**
Let $A = \{a, b, c, d, e\}$ and $B = \{1, 2, 3, 4\}$. Let the function mapping rules be:
* $f(a) = 2$, $f(b) = 1$, $f(c) = 4$, $f(d) = 1$, $f(e) = 1$

Find the image of the subset $S = \{b, c, d\}$.

* **Solution:** Check the mapping of each element in $S$:
  * $f(b) = 1$
  * $f(c) = 4$
  * $f(d) = 1$
* Gathering these unique elements into a subset yields:
  $$\mathbf{f(S) = \{1, 4\}}$$

### 🧠 Quick Check: Try it Yourself!

Let $f: \mathbf{Z} \rightarrow \mathbf{Z}$ be defined by $f(x) = 2x + 1$.
1. What is the **domain** and **codomain** of $f$?
2. If the subset $S = \{-1, 0, 1, 2\}$, what is the **image** $f(S)$?

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **Domain and Codomain:**
>    * **Domain:** $\mathbf{Z}$ (all integers).
>    * **Codomain:** $\mathbf{Z}$ (all integers).
>    * *Why?* The notation $f: \mathbf{Z} \rightarrow \mathbf{Z}$ explicitly defines the domain as the set before the arrow and the codomain as the set after the arrow.
> 2. **Image $f(S)$:** $\{-1, 1, 3, 5\}$.
>    * *Why?* Evaluate $f(x) = 2x + 1$ for each element in $S$:
>      * For $x = -1$: $f(-1) = 2(-1) + 1 = -1$
>      * For $x = 0$: $f(0) = 2(0) + 1 = 1$
>      * For $x = 1$: $f(1) = 2(1) + 1 = 3$
>      * For $x = 2$: $f(2) = 2(2) + 1 = 5$
>      * Gather the unique results: $f(S) = \{-1, 1, 3, 5\}$.
