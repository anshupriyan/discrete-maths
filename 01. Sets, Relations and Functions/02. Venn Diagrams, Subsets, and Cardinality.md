# Venn Diagrams, Subsets, and Cardinality (Sections 2.1.2 - 2.1.4)

---

### 1. Venn Diagrams (Section 2.1.2)

Venn diagrams are used to visualize sets geometrically.

* **The Universal Set ($U$):** This is the large container holding all possible objects under consideration for a given problem. In a Venn diagram, the universal set is represented by a **rectangle**.
* **Sets within the Universe:** Inside that rectangle, specific sets are represented by **circles** (or other closed geometric shapes).
* **Elements:** Individual elements are written as points inside the appropriate circles.

**Textbook Example:**
If the universal set is $U = \{a, e, i, o, u, \text{and sometimes } y\}$, and our set of standard vowels is $V = \{a, e, i, o, u\}$, a Venn diagram would show a rectangle labeled $U$, a circle inside labeled $V$ containing the letters $a, e, i, o, u$, and the letter $y$ sitting inside the rectangle but *outside* the circle.

### 2. Subsets (Section 2.1.3)

How sets relate to each other in terms of containment:

> **Definition 3:** The set $A$ is a **subset** of $B$ if and only if every element of $A$ is also an element of $B$.

* **Notation:** We write $A \subseteq B$ to state that $A$ is a subset of $B$.
* **Logical Condition:** To prove $A \subseteq B$, you must show that the conditional statement **if $x \in A$, then $x \in B$** is true for all elements in the universe. In quantifier notation:
  $$\forall x(x \in A \rightarrow x \in B)$$

#### Proving and Disproving Subsets

* **To show $A \subseteq B$:** You must pick an arbitrary element $x$ from $A$ and logically show it *must* belong to $B$.
* **To show $A \not\subseteq B$ (Not a subset):** You only need to find **one single counterexample**—that is, find one element $x$ such that $x \in A$ but $x \notin B$.

#### Two Vital Properties of Subsets:
For any set $S$:
1. **The Empty Set property:** $\emptyset \subseteq S$ for every set $S$.
   * *Why?* Because the conditional statement $\forall x(x \in \emptyset \rightarrow x \in S)$ is **vacuously true** (since $x \in \emptyset$ is always false).
2. **The Self-Containment property:** $S \subseteq S$ for every set $S$.
   * *Why?* Because $\forall x(x \in S \rightarrow x \in S)$ is a tautology.

### 3. Proper Subsets (Section 2.1.4)

If a set $A$ is a subset of $B$, it's possible that $A$ and $B$ are completely identical. If we want to specify that $B$ is strictly larger than $A$, we call it a **proper subset**.

* **Definition:** $A$ is a *proper subset* of $B$ if $A \subseteq B$ but $A \neq B$.
* **Notation:** This is written as $A \subset B$.
* **Meaning:** Every element in $A$ is in $B$, **and** there is at least one element in $B$ that does not exist in $A$.

---

### 4. The Size of a Set (Section 2.1.4 continued)

When we talk about the "size" of a set in discrete mathematics, we use a formal mathematical term: **cardinality**.

> **Definition 4:** Let $S$ be a set. If there are exactly $n$ distinct elements in $S$ where $n$ is a nonnegative integer, we say that $S$ is a **finite set** and that $n$ is the **cardinality** of $S$.

* **Notation:** The cardinality of $S$ is written as **$|S|$**.

#### Textbook Examples to Note:
* **Example 10:** Let $A$ be the set of odd positive integers less than 10 ($A = \{1, 3, 5, 7, 9\}$). There are exactly 5 distinct elements, so **$|A| = 5$**.
* **Example 11:** Let $S$ be the set of letters in the English alphabet. There are 26 letters, so **$|S| = 26$**.
* **Example 12:** Because the null/empty set ($\emptyset$) has absolutely no elements inside it, **$| \emptyset | = 0$**.

#### Infinite Sets
What happens if a set doesn't have a countable, fixed integer number of elements?

> **Definition 5:** A set is said to be **infinite** if it is not finite.

* **Example 13:** The set of all positive integers ($\mathbf{Z}^+$) is **infinite**. You can't assign a fixed integer $n$ to represent its total size.

### 🧠 Check for Understanding

#### Subsets Concept Check
Suppose we have the following three sets:
* $X = \{1, 2\}$
* $Y = \{1, 2, 3\}$
* $Z = \{1, 2, \{3\}\}$

Determine if the following statements are **True** or **False**:
1. $X \subseteq Y$
2. $X \subset Y$
3. $Y \subseteq Z$

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **True:** Every element of $X$ ($1$ and $2$) is also an element of $Y$. Thus, $X \subseteq Y$.
> 2. **True:** $X \subseteq Y$ and $X \neq Y$ (since $Y$ contains $3$, which is not in $X$). Thus, $X$ is a proper subset ($X \subset Y$).
> 3. **False:** The elements of $Y$ are `1`, `2`, and `3`. The elements of $Z$ are `1`, `2`, and the set `{3}`. The number `3` itself is not an element of $Z$ (only the nested set `{3}` is). Since $3 \in Y$ but $3 \notin Z$, $Y \not\subseteq Z$.
