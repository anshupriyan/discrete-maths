# Generalized Unions and Intersections (Section 2.2.3)

---

### 1. Moving Beyond Two Sets

Sets can be combined in collections larger than just two. Because the operations of union and intersection satisfy **associative laws**, expressions like $A \cup B \cup C$ and $A \cap B \cap C$ are perfectly well-defined and unambiguous:
$$A \cup (B \cup C) = (A \cup B) \cup C$$
$$A \cap (B \cap C) = (A \cap B) \cap C$$

* **The Union ($A \cup B \cup C$)** contains all elements that are in **at least one** of the sets $A$, $B$, or $C$.
* **The Intersection ($A \cap B \cap C$)** contains only the elements that are in **all three** sets $A$, $B$, and $C$.

#### Textbook Example 15:
Let:
* $A = \{0, 2, 4, 6, 8\}$
* $B = \{0, 1, 2, 3, 4\}$
* $C = \{0, 3, 6, 9\}$

Find $A \cup B \cup C$ and $A \cap B \cap C$.

* **Solution:**
  1. To find the **union**, gather every element that appears in *at least one* of the sets. Combining them (ignoring duplicates):
     $$A \cup B \cup C = \{0, 1, 2, 3, 4, 6, 8, 9\}$$
  2. To find the **intersection**, look for elements that are present in *all three* sets. The number $0$ is the only element that belongs to $A$, $B$, and $C$:
     $$A \cap B \cap C = \{0\}$$

### 2. Generalizing to $n$ Finite Sets

This concept generalizes to any finite collection of $n$ sets ($A_1, A_2, \dots, A_n$). We denote this using summation-like notations ($\bigcup$ and $\bigcap$):

> **Definition 6:** The **union** of a collection of sets is the set that contains those elements that are members of at least one set in the collection. To denote the union of the sets $A_1, A_2, \dots, A_n$, we write:
> $$A_1 \cup A_2 \cup \dots \cup A_n = \bigcup_{i=1}^{n} A_i$$

> **Definition 7:** The **intersection** of a collection of sets is the set that contains those elements that are members of all the sets in the collection. To denote the intersection of the sets $A_1, A_2, \dots, A_n$, we write:
> $$A_1 \cap A_2 \cap \dots \cap A_n = \bigcap_{i=1}^{n} A_i$$

#### Textbook Example 16:
For $i = 1, 2, 3, \dots$, let $A_i = \{i, i + 1, i + 2, \dots\}$. Find $\bigcup_{i=1}^{n} A_i$ and $\bigcap_{i=1}^{n} A_i$.

The first few sets in this sequence are:
* $A_1 = \{1, 2, 3, 4, \dots\}$
* $A_2 = \{2, 3, 4, 5, \dots\}$
* $A_3 = \{3, 4, 5, 6, \dots\}$
* ... up to $A_n = \{n, n+1, n+2, \dots\}$

* **Solution:**
  1. **For the Union ($\bigcup_{i=1}^{n} A_i$):** We need elements that are in *at least one* of these sets. Since $A_1$ already contains every positive integer starting from $1$, the union adds no additional elements:
     $$\bigcup_{i=1}^{n} A_i = A_1 \cup A_2 \cup \dots \cup A_n = \{1, 2, 3, \dots\} = A_1$$
  2. **For the Intersection ($\bigcap_{i=1}^{n} A_i$):** We need elements that belong to *every single set* from $A_1$ through $A_n$. Since numbers smaller than $n$ (like $1, 2, \dots, n-1$) are missing from $A_n$, they cannot be in the intersection. The first number that appears in *all* of them is $n$:
     $$\bigcap_{i=1}^{n} A_i = A_1 \cap A_2 \cap \dots \cap A_n = \{n, n + 1, n + 2, \dots\} = A_n$$

### 3. Generalizing to Infinite Families of Sets

We can extend this concept to include **infinite** collections of sets ($A_1, A_2, \dots, A_n, \dots$).

* To denote the union of an infinite family, we write:
  $$A_1 \cup A_2 \cup \dots \cup A_n \cup \dots = \bigcup_{i=1}^{\infty} A_i$$
* To denote the intersection of an infinite family, we write:
  $$A_1 \cap A_2 \cap \dots \cap A_n \cap \dots = \bigcap_{i=1}^{\infty} A_i$$

More broadly, for any index set $I$, we use existential ($\exists$) and universal ($\forall$) logical quantifiers to describe these collections precisely:
* $\bigcup_{i \in I} A_i = \{x \mid \exists i \in I \, (x \in A_i)\}$ *(An element is in the union if there exists at least one index $i$ where it belongs to $A_i$)*.
* $\bigcap_{i \in I} A_i = \{x \mid \forall i \in I \, (x \in A_i)\}$ *(An element is in the intersection if for all indices $i$, it belongs to $A_i$)*.

#### Textbook Example 17:
Suppose that $A_i = \{1, 2, 3, \dots, i\}$ for $i = 1, 2, 3, \dots$. Find $\bigcup_{i=1}^{\infty} A_i$ and $\bigcap_{i=1}^{\infty} A_i$.

The pattern of these sets is:
* $A_1 = \{1\}$
* $A_2 = \{1, 2\}$
* $A_3 = \{1, 2, 3\}$
* ... and so on.

* **Solution:**
  1. **For the Union ($\bigcup_{i=1}^{\infty} A_i$):**
     $$\bigcup_{i=1}^{\infty} A_i = \{1, 2, 3, \dots\} = \mathbf{Z}^+$$
     *Why?* Every positive integer $n$ is guaranteed to be in at least one of these sets, because it belongs directly to $A_n = \{1, 2, \dots, n\}$. Since every individual element across all sets is a positive integer, the final union is exactly the set of all positive integers ($\mathbf{Z}^+$).
  2. **For the Intersection ($\bigcap_{i=1}^{\infty} A_i$):**
     $$\bigcap_{i=1}^{\infty} A_i = \{1\}$$
     *Why?* The number $1$ is the only element that belongs to absolutely every single set in the sequence. We can verify this easily by looking at the very first set: $A_1 = \{1\}$. Since $A_1$ contains nothing but $1$, no other number can be shared by all sets.

### đź§  Quick Check: Try it Yourself!

Suppose we have the same set sequence: Let $A_i = \{1, 2, 3, \dots, i\}$ for $i = 1, 2, 3, \dots$.
Find the **finite** union and intersection up to $n$:
1. $\bigcup_{i=1}^{n} A_i$
2. $\bigcap_{i=1}^{n} A_i$

---

### đź’ˇ Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **$\bigcup_{i=1}^{n} A_i$:** $\{1, 2, 3, \dots, n\} = A_n$. 
>    * *Why?* The union collects all elements that belong to at least one set in the sequence $A_1, A_2, \dots, A_n$. Since each subsequent set is a superset of the previous ones (e.g. $A_1 \subset A_2 \subset \dots \subset A_n$), the largest set in the sequence ($A_n$) contains all elements present in the union.
> 2. **$\bigcap_{i=1}^{n} A_i$:** $\{1\} = A_1$.
>    * *Why?* The intersection collects elements that belong to all sets in the sequence $A_1, A_2, \dots, A_n$ simultaneously. The only element common to all of them is $1$. This can be checked by looking at the smallest set $A_1 = \{1\}$, which limits the intersection since no elements outside of $A_1$ can be present in all sets.
