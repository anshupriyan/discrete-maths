# Equivalence Relations (Sections 9.5 - 9.5.2)

---

A major milestone in discrete mathematics is using the mathematical properties of relations (reflexivity, symmetry, and transitivity) to group similar objects together. This process of clustering elements into coherent families is called partitioning, and it is governed by **equivalence relations**.

---

### 1. Introduction

In everyday life, we constantly group objects together based on a shared property:
* **Manufacturing:** Grouping all cars by their manufacturer (e.g., all Fords together, all Toyotas together).
* **Education:** Grouping school students together by their graduation year.
* **Computer Science:** Grouping data variables together by their data type (e.g., all integers in one class, all strings in another).

When we do this, we are saying that within a specific group, the elements are **equivalent** to each other based on that chosen rule.

---

### 2. Definition of an Equivalence Relation

To study these groupings mathematically, we use a special relation that passes a strict three-part test:

> **Definition 1:** A relation $R$ on a set $A$ is called an **equivalence relation** if it is:
> 1. **Reflexive**
> 2. **Symmetric**
> 3. **Transitive**

* **Notation:** Two elements $a$ and $b$ that are related by an equivalence relation are called **equivalent**. We write this as:
  $$a \sim b$$
  which indicates that $(a, b) \in R$.

---

### 3. Textbook Examples

Below are standard textbook cases demonstrating how to test relations systematically.

#### **TEXTBOOK EXAMPLE 1 (The String Length Relation)**
Let $A$ be the set of all strings of English letters. Let $R$ be the relation on $A$ defined by:
$$R = \{(a, b) \mid \text{string } a \text{ and string } b \text{ have the same length}\}$$
Is $R$ an equivalence relation?

**Solution:**
We must check all three properties systematically:
1. **Reflexive?** **Yes**. Every string $a$ clearly has the same length as itself:
   $$(a, a) \in R \quad \text{for all } a \in A$$
2. **Symmetric?** **Yes**. If string $a$ has the same length as string $b$, then string $b$ automatically has the same length as string $a$:
   $$\text{If } (a, b) \in R, \text{ then } (b, a) \in R$$
3. **Transitive?** **Yes**. If string $a$ has the same length as string $b$, and string $b$ has the same length as string $c$, then string $a$ must have the same length as string $c$:
   $$\text{If } (a, b) \in R \text{ and } (b, c) \in R, \text{ then } (a, c) \in R$$

Since $R$ is reflexive, symmetric, and transitive, it is a valid **equivalence relation**.

---

#### **TEXTBOOK EXAMPLE 3 (The Divides Comparison Failure)**
Let $R$ be the relation on the set of positive integers defined by:
$$R = \{(a, b) \mid a \text{ divides } b\}$$
*(i.e., $a \mid b$)*. Is $R$ an equivalence relation?

**Solution:**
Let's run our test:
1. **Reflexive?** **Yes**. Every integer $a$ divides itself:
   $$a \mid a \implies (a, a) \in R$$
2. **Symmetric?** **No!** For example, $2$ divides $4$, so $(2, 4) \in R$. However, $4$ does *not* divide $2$, so $(4, 2) \notin R$.

Since it fails the symmetry test, we can stop immediately. This relation is **not** an equivalence relation. *(Note: it is actually a partial ordering, which satisfies reflexivity, antisymmetry, and transitivity).*

### 🧠 Quick Check: Try it Yourself!

Let $R$ be the relation on the set of all real numbers $\mathbf{R}$ defined by:
$$R = \{(a, b) \mid a = b\}$$
*(This is the standard equals relation).*

Is this relation an equivalence relation? Check all 3 properties:
1. Is it reflexive? ($a = a$?)
2. Is it symmetric? (If $a = b$, does $b = a$?)
3. Is it transitive? (If $a = b$ and $b = c$, does $a = c$?)

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> * **Is it reflexive?** **Yes**.
>   * *Proof:* For any real number $a$, it is always true that $a = a$. Therefore, $(a, a) \in R$ for all $a \in \mathbf{R}$.
> * **Is it symmetric?** **Yes**.
>   * *Proof:* If $a$ is equal to $b$ ($a = b$), then it is mathematically identical to say $b$ is equal to $a$ ($b = a$). Therefore, if $(a, b) \in R$, then $(b, a) \in R$.
> * **Is it transitive?** **Yes**.
>   * *Proof:* If $a$ is equal to $b$ ($a = b$) and $b$ is equal to $c$ ($b = c$), then by the transitive property of equality, it must be that $a$ is equal to $c$ ($a = c$). Therefore, if $(a, b) \in R$ and $(b, c) \in R$, then $(a, c) \in R$.
> * **Conclusion:** Since the standard equality relation ($=$) is reflexive, symmetric, and transitive, it is a valid **equivalence relation** (the most fundamental equivalence relation in mathematics).
