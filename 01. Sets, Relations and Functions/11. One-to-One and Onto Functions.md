# One-to-One and Onto Functions (Section 2.3.2)

---

### 1. One-to-One (Injective) Functions

A function is **one-to-one**, or **injective**, if it never maps two different domain elements to the same codomain element.

> **Definition 5:** A function $f$ is said to be **one-to-one**, or **injective**, if and only if $f(a) = f(b)$ implies that $a = b$ for all $a$ and $b$ in the domain of $f$. A function is called an **injection** if it is one-to-one.

* **Conceptual Meaning:** If the outputs are equal, the inputs must be equal. Alternatively, distinct inputs must map to distinct outputs ($a \neq b \implies f(a) \neq f(b)$).

#### **Logical Quantification of Injections:**
$$\forall a \forall b \, (f(a) = f(b) \rightarrow a = b)$$

Or using the contrapositive:
$$\forall a \forall b \, (a \neq b \rightarrow f(a) \neq f(b))$$

#### **Textbook Example 8:**
Let $f$ be the function from $\{a, b, c, d\}$ to $\{1, 2, 3, 4, 5\}$ with:
* $f(a) = 4$, $f(b) = 5$, $f(c) = 1$, $f(d) = 3$

Is $f$ one-to-one?
* **Solution:** Yes. The function takes on distinct values for all elements in its domain; no output values are repeated.

#### **Textbook Example 9:**
Let $f(x) = x^2$ be a function from the set of integers $\mathbf{Z}$ to $\mathbf{Z}$. Is it one-to-one?
* **Solution:** No. A single counterexample disproves injection:
  * Take $x = 1$ and $x = -1$.
  * $1 \neq -1$, yet $f(1) = 1^2 = 1$ and $f(-1) = (-1)^2 = 1$.
  * Since $f(1) = f(-1)$ but $1 \neq -1$, $f$ is not injective.

#### **Textbook Example 10:**
Let $f(x) = x + 1$ be a function from the set of real numbers $\mathbf{R}$ to $\mathbf{R}$. Is it one-to-one?
* **Solution:** Yes. 
  * Assume $f(a) = f(b)$.
  * $a + 1 = b + 1$
  * Subtracting $1$ from both sides: $a = b$.
  * Because $f(a) = f(b) \implies a = b$, the function is injective.

---

### 2. Onto (Surjective) Functions

A function is **onto**, or **surjective**, if its range equals its codomain—meaning every element in the codomain has at least one preimage in the domain.

> **Definition 6:** A function $f$ from $A$ to $B$ is called **onto**, or **surjective**, if and only if for every element $b \in B$ there is an element $a \in A$ with $f(a) = b$. A function is called a **surjection** if it is onto.

#### **Logical Quantification of Surjections:**
$$\forall y \exists x \, (f(x) = y)$$
*(Where $x$ belongs to the domain $A$, and $y$ belongs to the codomain $B$)*

#### **Textbook Example 11:**
Let $f$ be the function from $\{a, b, c, d\}$ to $\{1, 2, 3\}$ defined by $f(a) = 3$, $f(b) = 2$, $f(c) = 1$, and $f(d) = 3$. Is $f$ onto?
* **Solution:** Yes. The codomain is $\{1, 2, 3\}$. Every element has a preimage:
  * $1$ is hit by $f(c)$
  * $2$ is hit by $f(b)$
  * $3$ is hit by $f(a)$ and $f(d)$
  * Since all codomain elements are mapped to, the function is surjective.

#### **Textbook Example 12:**
Let $f(x) = x^2$ be a function from the set of integers $\mathbf{Z}$ to $\mathbf{Z}$. Is it onto?
* **Solution:** No. Many integers in the codomain are not perfect squares (e.g., there is no integer $x$ such that $x^2 = -1$ or $x^2 = 2$). Since these elements have no preimages, the function is not surjective.

#### **Textbook Example 13:**
Let $f(x) = x + 1$ be a function from the set of integers $\mathbf{Z}$ to $\mathbf{Z}$. Is it onto?
* **Solution:** Yes. Let $y$ be any integer in the codomain $\mathbf{Z}$. We must find an integer $x$ in the domain such that $f(x) = y$:
  * Set $x + 1 = y \implies x = y - 1$.
  * Since $y \in \mathbf{Z}$, $y - 1$ is also guaranteed to be an integer. Thus, every possible integer output $y$ has a valid integer preimage $y - 1$.

---

### 3. One-to-One Correspondences (Bijections)

When a function is both one-to-one and onto, it creates a perfect, exclusive pairing between the domain and codomain sets.

> **Definition 7:** The function $f$ is a **one-to-one correspondence**, or a **bijection**, if it is both one-to-one and onto. A function is called a **bijection** if it is a one-to-one correspondence.

#### **Textbook Example 14:**
Let $f$ be the function from $\{a, b, c, d\}$ to $\{1, 2, 3, 4\}$ defined by $f(a) = 2$, $f(b) = 4$, $f(c) = 1$, and $f(d) = 3$. Is $f$ a bijection?
* **Solution:** Yes.
  1. It is **one-to-one** because all output values are distinct ($\{2, 4, 1, 3\}$).
  2. It is **onto** because the range $\{1, 2, 3, 4\}$ matches the codomain $\{1, 2, 3, 4\}$ exactly.
  * Since it satisfies both properties, it is a bijection.

---

### 4. Summary Checklist: Four Types of Mappings

1. **Injection only (One-to-one, not onto):** Distinct inputs map to distinct outputs, but some items in the codomain have no preimages.
2. **Surjection only (Onto, not one-to-one):** Every item in the codomain has a preimage, but multiple inputs point to the same output.
3. **Bijection (Both one-to-one and onto):** Perfect matching. Every element has exactly one unique partner.
4. **Neither:** Multiple inputs map to the same output *and* some items in the codomain have no preimages.

### 🧠 Quick Check: Try it Yourself!

Determine if the function $f: \mathbf{R} \rightarrow \mathbf{R}$ defined by $f(x) = 3x - 5$ is:
1. One-to-one (Injective)?
2. Onto (Surjective)?
3. A Bijection?

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **One-to-one (Injective)?** **Yes**.
>    * *Proof:* Assume $f(a) = f(b)$.
>      $$3a - 5 = 3b - 5 \implies 3a = 3b \implies a = b$$
>      Since assuming equal outputs directly proves equal inputs, $f$ is injective.
> 2. **Onto (Surjective)?** **Yes**.
>    * *Proof:* Let $y$ be any real number in the codomain $\mathbf{R}$. We must show there exists a real number $x$ such that $f(x) = y$:
>      $$3x - 5 = y \implies 3x = y + 5 \implies x = \frac{y + 5}{3}$$
>      Since $y \in \mathbf{R}$, the value $\frac{y + 5}{3}$ is also a real number. Thus, a valid preimage exists in the domain for every $y$ in the codomain.
> 3. **A Bijection?** **Yes**.
>    * *Proof:* Because the function $f$ is proven to be both injective (one-to-one) and surjective (onto), it is a bijection by definition.
