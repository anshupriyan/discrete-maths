# Inverse Functions and Composition of Functions (Section 2.3.3)

---

### 1. Inverse Functions

If a function $f$ takes an input $a$ from set $A$ and pairs it with an output $b$ from set $B$, the inverse function maps $b$ back to $a$.

#### **The Invertibility Requirement**
An inverse function can only be defined if the original function is a **one-to-one correspondence (a bijection)**:
* If $f$ is **not onto**, some element $b \in B$ has no preimage, leaving the inverse mapping rule undefined for that element.
* If $f$ is **not one-to-one**, multiple inputs map to the same target $b$, meaning the inverse rule would have to choose between multiple values, violating the definition of a function.

A function with a valid inverse is called **invertible**; otherwise, it is called **not invertible**.

> **Definition 9:** Let $f$ be a one-to-one correspondence from the set $A$ to the set $B$. The **inverse function of $f$** is the function that assigns to an element $b \in B$ the unique element $a \in A$ such that $f(a) = b$.
> 
> We denote this inverse function by **$f^{-1}$**. Therefore:
> $$f^{-1}(b) = a \quad \text{when} \quad f(a) = b$$

* **⚠️ Warning Note:** Do not confuse the function notation $f^{-1}$ with the algebraic reciprocal $1/f$. They are completely distinct concepts.

#### **Textbook Example 19 (Finite Mapping Inverse):**
Let $f$ be the function from $\{a, b, c\}$ to $\{1, 2, 3\}$ such that $f(a) = 2$, $f(b) = 3$, and $f(c) = 1$. Is $f$ invertible, and if so, what is its inverse?
* **Solution:** Yes. $f$ is invertible because it is a bijection. Reversing the mappings yields:
  * **$f^{-1}(1) = c$** (since $f(c) = 1$)
  * **$f^{-1}(2) = a$** (since $f(a) = 2$)
  * **$f^{-1}(3) = b$** (since $f(b) = 3$)

#### **Textbook Example 20 (Integer Algebra Inverse):**
Let $f: \mathbf{Z} \rightarrow \mathbf{Z}$ be defined by $f(x) = x + 1$. Is $f$ invertible, and if so, what is its inverse?
* **Solution:** Yes, $f(x) = x + 1$ is a bijection. Assign the output to $y$ and solve for $x$:
  $$y = x + 1 \implies x = y - 1$$
  Thus, the inverse function is:
  $$\mathbf{f^{-1}(y) = y - 1}$$

#### **Textbook Example 22 (Restricting Domains):**
The squaring function $f(x) = x^2$ over all real numbers $\mathbf{R}$ is not invertible because it is not one-to-one ($f(-2) = f(2) = 4$).

However, if we restrict **both the domain and codomain to nonnegative real numbers** ($[0, \infty)$), the function becomes invertible:
* It is **one-to-one** because $x^2 = y^2 \implies (x+y)(x-y) = 0$. Since inputs are non-negative, $x+y=0 \implies x=y=0$. Otherwise, $x-y=0 \implies x=y$.
* It is **onto** because every nonnegative number has a valid real square root ($x = \sqrt{y}$).
* Thus, the inverse function for this restricted domain is:
  $$\mathbf{f^{-1}(y) = \sqrt{y}}$$

### 2. Composition of Functions

When the output of one function serves as the input for a second function, they can be combined into a single operation called a **composition**.

> **Definition 10:** Let $g$ be a function from set $A$ to set $B$, and let $f$ be a function from set $B$ to set $C$. The **composition of the functions $f$ and $g$**, denoted by **$f \circ g$**, is defined for all $a \in A$ by:
> $$(f \circ g)(a) = f(g(a))$$

* **Order of Execution:** In $f \circ g$, **$g$ is evaluated first**, and then $f$ is applied to that result.
* **Domain Constraint:** A composition is only defined if the range of the first function is a subset of the domain of the second function ($\text{Range}(g) \subseteq \text{Domain}(f)$).

#### **Textbook Example 23 (Discrete Set Composition):**
Let $g: \{a, b, c\} \rightarrow \{a, b, c\}$ be defined by $g(a) = b, g(b) = c, g(c) = a$.
Let $f: \{a, b, c\} \rightarrow \{1, 2, 3\}$ be defined by $f(a) = 3, f(b) = 2, f(c) = 1$.
Find $f \circ g$ and $g \circ f$ if they are defined.

* **Solution:**
  1. **For $f \circ g$:** Route each input through $g$, then $f$:
     * $(f \circ g)(a) = f(g(a)) = f(b) = \mathbf{2}$
     * $(f \circ g)(b) = f(g(b)) = f(c) = \mathbf{1}$
     * $(f \circ g)(c) = f(g(c)) = f(a) = \mathbf{3}$
  2. **For $g \circ f$:** This composition **is not defined** because the range of $f$ ($\{1, 2, 3\}$) is not contained in the domain of $g$ ($\{a, b, c\}$).

#### **Textbook Example 24 (Algebraic Composition):**
Let $f(x) = 2x + 3$ and $g(x) = 3x + 2$ be functions from $\mathbf{Z}$ to $\mathbf{Z}$. Find $f \circ g$ and $g \circ f$.

* **Solution:**
  1. **$f \circ g$:** Substitute $g(x)$ into $f$:
     $$(f \circ g)(x) = f(g(x)) = f(3x + 2) = 2(3x + 2) + 3 = \mathbf{6x + 7}$$
  2. **$g \circ f$:** Substitute $f(x)$ into $g$:
     $$(g \circ f)(x) = g(f(x)) = g(2x + 3) = 3(2x + 3) + 2 = \mathbf{6x + 11}$$
* **⚠️ Crucial Takeaway:** Since $6x + 7 \neq 6x + 11$, **composition is not commutative** ($f \circ g \neq g \circ f$).

---

### 3. Composing a Function with its Inverse

Composing a valid bijective function directly with its own inverse reverses the mapping, resulting in an **identity function** ($\iota$):
$$(f^{-1} \circ f)(a) = f^{-1}(f(a)) = f^{-1}(b) = a = \iota_A(a)$$
$$(f \circ f^{-1})(b) = f(f^{-1}(b)) = f(a) = b = \iota_B(b)$$

### 🧠 Quick Check: Try it Yourself!

Let $f: \mathbf{R} \rightarrow \mathbf{R}$ be defined by $f(x) = 4x - 7$.
1. Find the inverse function rule $f^{-1}(y)$.
2. If $g(x) = x^2$, write down the composition formula for $(g \circ f)(x)$.

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> 1. **Inverse Function $f^{-1}(y)$:** $\frac{y + 7}{4}$.
>    * *Proof:* Set the function output equal to $y$ and solve algebraically for $x$:
>      $$y = 4x - 7 \implies y + 7 = 4x \implies x = \frac{y + 7}{4}$$
>      Thus, the inverse function mapping is $f^{-1}(y) = \frac{y + 7}{4}$.
> 2. **Composition $(g \circ f)(x)$:** $(4x - 7)^2 = 16x^2 - 56x + 49$.
>    * *Proof:* Substitute $f(x)$ into the outer function $g(x)$:
>      $$(g \circ f)(x) = g(f(x)) = g(4x - 7) = (4x - 7)^2$$
>      Expanding the binomial:
>      $$(4x - 7)^2 = (4x)^2 - 2(4x)(7) + (-7)^2 = 16x^2 - 56x + 49$$
