# Representing Relations (Sections 9.3 - 9.3.2)

---

While defining relations as sets of ordered pairs is mathematically precise, it can become unwieldy for large sets. To store and analyze these relations efficiently, we use alternative structures. In computer memory, we represent relations using **zero-one matrices** (bit matrices).

---

### 1. Representing Relations Using Matrices

A **zero-one matrix** (a matrix whose entries are strictly 0 or 1) provides a systematic way to catalog which elements in a domain are related to elements in a codomain.

> **Definition 1:** Suppose $R$ is a relation from $A = \{a_1, a_2, \dots, a_m\}$ to $B = \{b_1, b_2, \dots, b_n\}$. The matrix representing $R$ is the $m \times n$ matrix:
> $$M_R = [m_{ij}]$$
> where:
> * $m_{ij} = 1$ if $(a_i, b_j) \in R$
> * $m_{ij} = 0$ if $(a_i, b_j) \notin R$

In this matrix, rows represent the elements of the domain $A$, and columns represent the elements of the codomain $B$.

#### **Textbook Example 1:**
Let $A = \{1, 2, 3\}$ and $B = \{1, 2\}$. Let $R$ be the relation from $A$ to $B$ containing $(2, 1), (3, 1),$ and $(3, 2)$. What is the matrix $M_R$?

* **Solution:** We set up a grid with rows representing $A$ ($1, 2, 3$) and columns representing $B$ ($1, 2$):

|  | 1 | 2 |
| :--- | :---: | :---: |
| **1** | 0 | 0 |
| **2** | 1 | 0 |
| **3** | 1 | 1 |

Thus, the zero-one matrix representing $R$ is:
$$M_R = \begin{bmatrix} 0 & 0 \\ 1 & 0 \\ 1 & 1 \end{bmatrix}$$

---

### 2. Analyzing Properties via Matrices

Representing relations as matrices allows us to determine key relational properties (such as reflexivity and symmetry) simply by inspecting the matrix structure:

1. **Reflexive:** A relation $R$ on a set $A$ is reflexive if and only if the main diagonal entries (where $i = j$) are all 1s:
   $$m_{ii} = 1 \quad \text{for all } i = 1, 2, \dots, n$$
2. **Symmetric:** A relation $R$ is symmetric if and only if the matrix is symmetric across the main diagonal. This means the matrix is equal to its transpose:
   $$M_R = M_R^T$$
   *(i.e., $m_{ij} = m_{ji}$ for all $i, j$)*.

#### **Textbook Example 2:**
Is the relation represented by the matrix $M_R = \begin{bmatrix} 1 & 1 & 0 \\ 1 & 1 & 0 \\ 0 & 0 & 1 \end{bmatrix}$ reflexive and symmetric?

* **Solution:**
  * **Reflexive?** **Yes**. The main diagonal entries $m_{11}$, $m_{22}$, and $m_{33}$ are all 1.
  * **Symmetric?** **Yes**. The matrix is symmetric across the main diagonal ($m_{12} = m_{21} = 1$, and all other non-diagonal entries are 0).

---

### 🧠 Quick Check: Try it Yourself!

Let $A = \{1, 2\}$. Write the matrix $M_R$ for the relation:
$$R = \{(1, 1), (1, 2), (2, 2)\}$$

Is this relation reflexive? Is it symmetric?

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> * **Zero-One Matrix $M_R$:**
>   $$M_R = \begin{bmatrix} 1 & 1 \\ 0 & 1 \end{bmatrix}$$
>   * *Proof:* We construct a $2 \times 2$ grid:
>     * Row 1, Col 1 representing $(1, 1) \in R \rightarrow m_{11} = 1$
>     * Row 1, Col 2 representing $(1, 2) \in R \rightarrow m_{12} = 1$
>     * Row 2, Col 1 representing $(2, 1) \notin R \rightarrow m_{21} = 0$
>     * Row 2, Col 2 representing $(2, 2) \in R \rightarrow m_{22} = 1$
> * **Is it Reflexive?** **Yes**.
>   * *Proof:* The main diagonal entries $m_{11}$ and $m_{22}$ are both 1.
> * **Is it Symmetric?** **No**.
>   * *Proof:* The entry $m_{12} = 1$ but the entry $m_{21} = 0$. Since $M_R \neq M_R^T$ (the matrix is not symmetric across its main diagonal), the relation is not symmetric.
