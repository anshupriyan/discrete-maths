# Summations (Section 2.4.5)

---

### 1. The Summation Notation

To denote the sum of consecutive terms $a_m, a_{m+1}, \dots, a_n$ from a sequence $\{a_n\}$, we use the uppercase Greek letter **Sigma** ($\sum$):

$$\sum_{j=m}^{n} a_j \quad \text{or} \quad \sum_{m \le j \le n} a_j$$

This represents:
$$a_m + a_{m+1} + \dots + a_n$$

#### **Core Terminology:**
* **Index of Summation ($j$):** The variable that increments through each consecutive integer. The choice of letter is arbitrary (e.g., $i$, $j$, or $k$):
  $$\sum_{j=m}^{n} a_j = \sum_{i=m}^{n} a_i = \sum_{k=m}^{n} a_k$$
* **Lower Limit ($m$):** The starting integer index.
* **Upper Limit ($n$):** The ending integer index.

The index increments through every single consecutive integer from $m$ to $n$ inclusive.

### 2. Textbook Examples

#### **Textbook Example 17 (Writing the Summation):**
Use summation notation to express the sum of the first 100 terms of the sequence $\{a_j\}$, where $a_j = 1/j$ for $j = 1, 2, 3, \dots$.
* **Solution:** The lower limit is 1, and the upper limit is 100. We place the rule inside the operator:
  $$\mathbf{\sum_{j=1}^{100} \frac{1}{j}}$$

#### **Textbook Example 18 (Evaluating a Numeric Sum):**
What is the value of $\sum_{j=1}^{5} j^2$?
* **Solution:** Sequentially evaluate $j = 1, 2, 3, 4, 5$ and add them together:
  $$\sum_{j=1}^{5} j^2 = 1^2 + 2^2 + 3^2 + 4^2 + 5^2$$
  $$= 1 + 4 + 9 + 16 + 25 = \mathbf{55}$$

#### **Textbook Example 19 (Alternating Sign Limit):**
What is the value of $\sum_{k=4}^{8} (-1)^k$?
* **Solution:** Run the index through $k = 4, 5, 6, 7, 8$:
  $$\sum_{k=4}^{8} (-1)^k = (-1)^4 + (-1)^5 + (-1)^6 + (-1)^7 + (-1)^8$$
  An even power turns $-1$ into $1$, and an odd power leaves it as $-1$:
  $$= 1 + (-1) + 1 + (-1) + 1 = \mathbf{1}$$

### 3. Shifting Indices

Sometimes you need to change the starting and ending index variables to align two equations. When you shift the boundaries, you must balance the internal variable accordingly so the actual expanded calculation does not alter.

#### **Textbook Example 20:**
Suppose we have the sum $\sum_{j=1}^{5} j^2$, but want the index of summation to run between $0$ and $4$ instead.
* **Solution:** Set a new variable $k = j - 1$. Let's compute the new boundaries:
  * **Lower limit:** When $j = 1$, $k = 1 - 1 = 0$.
  * **Upper limit:** When $j = 5$, $k = 5 - 1 = 4$.
  * **Variable Substitution:** Solve for the original variable: $j = k + 1$. Thus, the term $j^2$ becomes $(k + 1)^2$.
  $$\mathbf{\sum_{j=1}^{5} j^2 = \sum_{k=0}^{4} (k+1)^2}$$
  *(Expanding this still outputs $1^2 + 2^2 + 3^2 + 4^2 + 5^2 = 55$, verifying the shift worked).*

### 4. Important Summation Formulas (Table 2)

The textbook introduces extremely useful closed-form summation shortcuts. Instead of manually adding thousands of terms, you can use these formulas to calculate the result instantly:

1. **Geometric Series:**
   $$\sum_{k=0}^{n} a r^k = \frac{a r^{n+1} - a}{r - 1} \quad (r \neq 1)$$
2. **Sum of First $n$ Integers:**
   $$\sum_{k=1}^{n} k = \frac{n(n + 1)}{2}$$
3. **Sum of Squares:**
   $$\sum_{k=1}^{n} k^2 = \frac{n(n + 1)(2n + 1)}{6}$$
4. **Sum of Cubes:**
   $$\sum_{k=1}^{n} k^3 = \frac{n^2(n + 1)^2}{4}$$

#### **Textbook Example 23 (Using the Formulas):**
Find $\sum_{k=50}^{100} k^2$.
* **Solution:** Since the sum does not start at 1, we express it as the difference between two sums that *do* start at 1:
  $$\sum_{k=50}^{100} k^2 = \sum_{k=1}^{100} k^2 - \sum_{k=1}^{49} k^2$$
  Now apply the formula $\frac{n(n + 1)(2n + 1)}{6}$ to both parts:
  * **For $n = 100$:** $\frac{100 \cdot 101 \cdot 201}{6} = 338,350$.
  * **For $n = 49$:** $\frac{49 \cdot 50 \cdot 99}{6} = 40,425$.
  Subtracting the two values gives the final result:
  $$338,350 - 40,425 = \mathbf{297,925}$$

### 5. Double Summations

A **double summation** features an outer sum and an inner sum, which frequently models nested loop structures in programming. To evaluate them, **work from the inside out**: first expand and solve the inner summation completely, then evaluate the remaining outer summation on those results.

#### **Textbook Example 21:**
Evaluate the double summation $\sum_{i=1}^{4} \sum_{j=1}^{3} ij$.
* **Solution:**
  1. **Step 1: Evaluate the inner sum** ($\sum_{j=1}^{3} ij$) while treating $i$ as a temporary constant:
     $$\sum_{j=1}^{3} ij = (i \cdot 1) + (i \cdot 2) + (i \cdot 3) = 6i$$
  2. **Step 2: Substitute this result back into the outer sum** ($\sum_{i=1}^{4}$):
     $$\sum_{i=1}^{4} 6i = 6(1) + 6(2) + 6(3) + 6(4)$$
     $$= 6 + 12 + 18 + 24 = \mathbf{60}$$

### 🧠 Quick Check: Try it Yourself!

Find the exact numerical value of this summation:
$$\sum_{k=1}^{5} (2k + 3)$$

---

### 💡 Solutions & Explanation

> [!NOTE]
> Here are the step-by-step verification answers for the check above:
> 
> * **Answer:** **$45$**.
> * **Method A (Direct Expansion):** Evaluate the terms for $k = 1, 2, 3, 4, 5$:
>   * $k = 1: 2(1) + 3 = 5$
>   * $k = 2: 2(2) + 3 = 7$
>   * $k = 3: 2(3) + 3 = 9$
>   * $k = 4: 2(4) + 3 = 11$
>   * $k = 5: 2(5) + 3 = 13$
>   Summing these terms: $5 + 7 + 9 + 11 + 13 = \mathbf{45}$.
> * **Method B (Using Summation Formula Shortcuts):** Split the summation using linearity:
>   $$\sum_{k=1}^{5} (2k + 3) = 2 \sum_{k=1}^{5} k + \sum_{k=1}^{5} 3$$
>   Using the formulas:
>   * $2 \sum_{k=1}^{5} k = 2 \left( \frac{5(5 + 1)}{2} \right) = 2(15) = 30$
>   * $\sum_{k=1}^{5} 3 = 3 \times 5 = 15$
>   Adding them: $30 + 15 = \mathbf{45}$.
