### Question 1(a)
**Question:**
Define a Hamilton path and Hamilton circuit. List a Hamilton path or Hamilton circuit in the graphs G1 and G2 given below if it exists.

*(G1 and G2 are the graphs shown in the exam paper Page 2)*
* **G1:** A pentagon-like or house-like graph with vertices $a, b, c, d, e$, where $a-b-c-d-e-a$ forms a cycle, and diagonals $(a, c)$ and $(b, e)$ cross, and $d$ is connected to $c$ and $e$.
* **G2:** A path of 4 vertices: $a - b - d - c$.

---

#### Solution:

1. **Definitions:**
   * **Hamilton Path:** A simple path in a graph $G$ that passes through every vertex in $G$ exactly once.
   * **Hamilton Circuit:** A simple circuit in a graph $G$ that passes through every vertex in $G$ exactly once (starting and ending at the same vertex).

2. **Analysis of Graph G1:**
   * **Hamilton Circuit:** Yes, G1 has a Hamilton circuit.
     * Let's trace the path: $e \to d \to c \to a \to b \to e$.
     * This path visits the vertices $e, d, c, a, b$ exactly once before returning to the start vertex $e$. All transitions are valid edges in G1.
     * Thus, a Hamilton circuit is: **$e - d - c - a - b - e$** (or any cyclic shift).
   * **Hamilton Path:** Since it has a Hamilton circuit, we can obtain a Hamilton path by removing the last edge: **$e - d - c - a - b$**.

3. **Analysis of Graph G2:**
   * **Hamilton Path:** Yes, G2 is a simple path.
     * The path is: **$a - b - d - c$** (it visits every vertex exactly once).
   * **Hamilton Circuit:** No, G2 does not contain a Hamilton circuit.
     * Vertices $a$ and $c$ have degree 1.
     * A Hamilton circuit requires every vertex to have degree at least 2 in the circuit (since the path must enter and leave each vertex).
     * Thus, no Hamilton circuit can exist in a graph containing a vertex of degree 1.

### Question 1(b)
**Question:**
(i) Find the contrapositive, the converse and the inverse of the statement:
"The home team wins whenever it is raining."
(ii) Show that if $n$ is an integer and $3n + 2$ is odd, then $n$ is odd using a proof by contraposition.

---

#### Solution:

1. **(i) Statement Variants:**
   * Let the original statement be parsed: "The home team wins ($q$) whenever it is raining ($p$)." This translates to the conditional statement $p \rightarrow q$: "If it is raining, then the home team wins."
   * **Contrapositive ($\neg q \rightarrow \neg p$):** "If the home team does not win, then it is not raining."
   * **Converse ($q \rightarrow p$):** "If the home team wins, then it is raining."
   * **Inverse ($\neg p \rightarrow \neg q$):** "If it is not raining, then the home team does not win."

2. **(ii) Proof by Contraposition:**
   * We want to prove: "If $3n + 2$ is odd, then $n$ is odd."
   * The **contrapositive statement** is: "If $n$ is even (not odd), then $3n + 2$ is even (not odd)."
   * **Direct Proof of the Contrapositive:**
     * Assume $n$ is an even integer.
     * By definition, $n = 2k$ for some integer $k$.
     * Substitute $n$ into the expression $3n + 2$:
       $$3n + 2 = 3(2k) + 2 = 6k + 2 = 2(3k + 1)$$
     * Let $j = 3k + 1$. Since $k$ is an integer, $j$ is also an integer.
     * Thus, $3n + 2 = 2j$, which means $3n + 2$ is an even integer by definition.
   * Since the contrapositive statement is true, the original statement is also true.

### Question 1(c)
**Question:**
Find the greatest common divisor of 414 and 662 using the Euclidean algorithm.

---

#### Solution:
To find $\gcd(414, 662)$ using the Euclidean algorithm, we perform successive divisions:

1. **Step 1:** Divide $662$ by $414$:
   $$662 = 1 \times 414 + 248$$

2. **Step 2:** Divide $414$ by the remainder $248$:
   $$414 = 1 \times 248 + 166$$

3. **Step 3:** Divide $248$ by the remainder $166$:
   $$248 = 1 \times 166 + 82$$

4. **Step 4:** Divide $166$ by the remainder $82$:
   $$166 = 2 \times 82 + 2$$

5. **Step 5:** Divide $82$ by the remainder $2$:
   $$82 = 41 \times 2 + 0$$

Since the remainder is $0$, the last non-zero remainder is $2$.
Therefore, $\gcd(414, 662) = 2$.

### Question 1(d)
**Question:**
Let $f$ be the function from the set $X = \{2, 3, 4, 5, 6, 7\}$ into the set $Y = \{0, 1, 2, 3, 4\}$ defined by $f(x) = 2x \pmod 5$. Write $f$ as a set of ordered pairs. Is $f$ one-to-one or onto? Justify.

---

#### Solution:

1. **Compute $f(x)$ for each $x \in X$:**
   * $f(2) = 2(2) \bmod 5 = 4$
   * $f(3) = 2(3) \bmod 5 = 6 \bmod 5 = 1$
   * $f(4) = 2(4) \bmod 5 = 8 \bmod 5 = 3$
   * $f(5) = 2(5) \bmod 5 = 10 \bmod 5 = 0$
   * $f(6) = 2(6) \bmod 5 = 12 \bmod 5 = 2$
   * $f(7) = 2(7) \bmod 5 = 14 \bmod 5 = 4$

2. **Represent $f$ as a set of ordered pairs:**
   $$f = \{ (2, 4), (3, 1), (4, 3), (5, 0), (6, 2), (7, 4) \}$$

3. **Is $f$ One-to-One (Injective)?**
   * **Answer:** **No**.
   * **Justification:** A function is one-to-one if distinct inputs map to distinct outputs. Here, $2 \neq 7$, but $f(2) = f(7) = 4$. Therefore, $f$ is not one-to-one.

4. **Is $f$ Onto (Surjective)?**
   * **Answer:** **Yes**.
   * **Justification:** A function is onto if the range equals the codomain.
     * The codomain is $Y = \{0, 1, 2, 3, 4\}$.
     * The range is $\text{Range}(f) = \{f(2), f(3), f(4), f(5), f(6), f(7)\} = \{4, 1, 3, 0, 2\} = \{0, 1, 2, 3, 4\}$.
     * Since $\text{Range}(f) = Y$, every element in $Y$ has at least one pre-image in $X$. Thus, $f$ is onto.

   * Therefore, the minimum number of students required is:

### Question 1(f)
**Question:**
Let $a$ be a numeric function such that:
$$a_r = \begin{cases} 1 & 0 \le r \le 3 \\ 2^r + 3 & r \ge 4 \end{cases}$$
Find $\Delta a$ and $S^{-2} a$.

---

#### Solution:

1. **Find $\Delta a$ (Forward Difference Operator):**
   * By definition: $(\Delta a)_r = a_{r+1} - a_r$ for $r \ge 0$.
     * **Case 1: $0 \le r \le 2$**
       $$(\Delta a)_r = 1 - 1 = 0$$
     * **Case 2: $r = 3$**
       $$a_3 = 1$$
       $$a_4 = 2^4 + 3 = 19$$
       $$(\Delta a)_3 = a_4 - a_3 = 19 - 1 = 18$$
     * **Case 3: $r \ge 4$**
       $$(\Delta a)_r = \left(2^{r+1} + 3\right) - \left(2^r + 3\right) = 2^{r+1} - 2^r = 2^r(2 - 1) = 2^r$$
   * Combined:
     $$(\Delta a)_r = \begin{cases}
     0 & 0 \le r \le 2 \\
     18 & r = 3 \\
     2^r & r \ge 4
     \end{cases}$$

2. **Find $S^{-2} a$ (Backward Shifting Operator):**
   * By definition, a negative shift index $S^{-k}$ shifts the function backward (shifts left):
     $$(S^{-k} a)_r = a_{r+k}$$
   * For $k = 2$:
     $$(S^{-2} a)_r = a_{r+2} \quad \text{for } r \ge 0$$
   * Substitute values:
     * If $0 \le r+2 \le 3 \implies 0 \le r \le 1$:
       $$(S^{-2} a)_r = 1$$
     * If $r+2 \ge 4 \implies r \ge 2$:
       $$(S^{-2} a)_r = 2^{r+2} + 3$$
   * Combined:
     $$(S^{-2} a)_r = \begin{cases}
     1 & r = 0, 1 \\
     2^{r+2} + 3 & r \ge 2
     \end{cases}$$

*(For completeness, if the print was a typo for $S^2 a$:)*
$$(S^2 a)_r = \begin{cases} 0 & r=0,1 \\ 1 & 2 \le r \le 5 \\ 2^{r-2} + 3 & r \ge 6 \end{cases}$$

### Question 2(b)
**Question:**
Let $R$ be a relation on the set of real numbers such that $R = \{(a, b) \mid a^2 + b^2 = 1\}$. State whether the relation $R$ is reflexive, symmetric, antisymmetric and transitive. Briefly justify your answer in each case.

---

#### Solution:
The relation $R$ is defined on the set of real numbers $\mathbb{R}$ by:
$$a R b \iff a^2 + b^2 = 1$$

1. **Reflexive:**
   * **Answer:** **No**.
   * **Justification:** $R$ is reflexive if $x^2 + x^2 = 1 \implies 2x^2 = 1 \implies x = \pm 1/\sqrt{2}$ for all $x \in \mathbb{R}$.
     * If we pick $x = 2 \in \mathbb{R}$, then $2^2 + 2^2 = 8 \neq 1$.
     * Thus, $(2, 2) \notin R$. So $R$ is not reflexive.

2. **Symmetric:**
   * **Answer:** **Yes**.
   * **Justification:** $R$ is symmetric if for all $a, b \in \mathbb{R}$, $(a, b) \in R \implies (b, a) \in R$.
     * Assume $(a, b) \in R \implies a^2 + b^2 = 1$.
     * Since addition is commutative on real numbers, $b^2 + a^2 = 1$.
     * Thus, $(b, a) \in R$. So $R$ is symmetric.

3. **Antisymmetric:**
   * **Answer:** **No**.
   * **Justification:** $R$ is antisymmetric if $(a, b) \in R$ and $(b, a) \in R \implies a = b$.
     * Let $a = 0$ and $b = 1$.
     * $a^2 + b^2 = 0^2 + 1^2 = 1 \implies (0, 1) \in R$.
     * $b^2 + a^2 = 1^2 + 0^2 = 1 \implies (1, 0) \in R$.
     * However, $0 \neq 1$. So $R$ is not antisymmetric.

4. **Transitive:**
   * **Answer:** **No**.
   * **Justification:** $R$ is transitive if $(a, b) \in R$ and $(b, c) \in R \implies (a, c) \in R$.
     * Let $a = 1, b = 0,$ and $c = -1$.
     * $(1, 0) \in R$ because $1^2 + 0^2 = 1$.
     * $(0, -1) \in R$ because $0^2 + (-1)^2 = 1$.
     * However, $(1, -1) \notin R$ because $1^2 + (-1)^2 = 2 \neq 1$.
     * So $R$ is not transitive.

### Question 2(c)
**Question:**
Use mathematical induction to prove that $n^3 + 2n$ is divisible by 3 for every integer $n \ge 1$.

---

#### Solution:
Let $P(n)$ be the statement: "3 divides $n^3 + 2n$." (i.e. $n^3 + 2n = 3j$ for some $j \in \mathbb{Z}$).

1. **Basis Step ($n = 1$):**
   * For $n = 1$:
     $$n^3 + 2n = 1^3 + 2(1) = 3$$
   * Since 3 is divisible by 3 ($3 = 3 \times 1$), $P(1)$ is true.

2. **Inductive Step:**
   * Assume $P(k)$ is true for some positive integer $k \ge 1$ (**Inductive Hypothesis**):
     $$k^3 + 2k = 3m \quad \text{for some } m \in \mathbb{Z}$$
   * We need to show that $P(k+1)$ is true: "3 divides $(k+1)^3 + 2(k+1)$."
   * Expand the expression for $n = k+1$:
     $$(k+1)^3 + 2(k+1) = \left(k^3 + 3k^2 + 3k + 1\right) + (2k + 2)$$
   * Regroup terms to isolate the inductive hypothesis component:
     $$= \left(k^3 + 2k\right) + 3k^2 + 3k + 3$$
   * Substitute the inductive hypothesis ($k^3 + 2k = 3m$):
     $$= 3m + 3\left(k^2 + k + 1\right) = 3\left(m + k^2 + k + 1\right)$$
   * Since $m$ and $k$ are integers, $m + k^2 + k + 1$ is also an integer (say, $j$).
   * Thus, $(k+1)^3 + 2(k+1) = 3j$, which means it is divisible by 3.
   * Therefore, $P(k+1)$ is true.

**Conclusion:**
By the principle of mathematical induction, $n^3 + 2n$ is divisible by 3 for all integers $n \ge 1$.

### Question 3(b)
**Question:**
Find the sum and the product of $(1001)_2$ and $(1101)_2$ without using decimal number system.

---

#### Solution:

1. **Calculate the Sum (Binary Addition):**
   ```
     [Carry]  1 0 0 1
              1 0 0 1
            + 1 1 0 1
            ---------
            1 0 1 1 0
   ```
   * Step-by-step:
     * $1 + 1 = 10_2 \implies 0$ (carry 1)
     * $0 + 0 + 1\text{ (carry)} = 1$ (carry 0)
     * $0 + 1 = 1$ (carry 0)
     * $1 + 1 = 10_2 \implies 0$ (carry 1)
     * Bring down carry: $1$.
   * **Sum:** **$(10110)_2$** (which is $22$ in decimal).

2. **Calculate the Product (Binary Multiplication):**
   ```
              1 0 0 1
            x 1 1 0 1
            ---------
              1 0 0 1   (1001 x 1)
            0 0 0 0     (1001 x 0, shifted)
          1 0 0 1       (1001 x 1, shifted twice)
        + 1 0 0 1       (1001 x 1, shifted thrice)
        -------------
        1 1 1 0 1 0 1
   ```
   * Summing columns:
     * Column 0: $1$
     * Column 1: $0$
     * Column 2: $0 + 0 + 1 = 1$
     * Column 3: $1 + 0 + 0 + 1 = 10_2 \implies 0$ (carry 1)
     * Column 4: $0 + 0 + 0 + 1\text{ (carry)} = 1$ (carry 0)
     * Column 5: $1 + 0 = 1$
     * Column 6: $1$
   * **Product:** **$(1110101)_2$** (which is $117$ in decimal).

### Question 3(c)
**Question:**
In a survey of 60 people, it was found that 25 people read Hindi newspaper, 26 read English newspaper, 26 read Urdu newspaper, 9 read both Hindi and Urdu newspaper, 11 read both Hindi and English newspaper, 8 read both English and Urdu newspaper, 3 read all three newspapers. Find :
(i) The number of people who read at least one of the three newspapers.
(ii) The number of people who read exactly one newspaper.
(iii) The number of people who read both Hindi and Urdu newspaper, but not English?

---

#### Solution:
Let $H, E, U$ be the sets of people who read Hindi, English, and Urdu newspapers respectively. We are given:
* Total surveyed $= 60$
* $|H| = 25, \quad |E| = 26, \quad |U| = 26$
* $|H \cap E| = 11, \quad |H \cap U| = 9, \quad |E \cap U| = 8$
* $|H \cap E \cap U| = 3$

1. **(i) Number of people who read at least one newspaper ($|H \cup E \cup U|$):**
   * By the **Principle of Inclusion-Exclusion**:
     $$|H \cup E \cup U| = |H| + |E| + |U| - |H \cap E| - |H \cap U| - |E \cap U| + |H \cap E \cap U|$$
     $$|H \cup E \cup U| = 25 + 26 + 26 - 11 - 9 - 8 + 3 = 77 - 28 + 3 = \mathbf{52 \text{ people}}$$

2. **(ii) Number of people who read exactly one newspaper:**
   * Let's calculate the disjoint regions of the Venn diagram:
     * **Only Hindi:** $|H| - |H \cap E| - |H \cap U| + |H \cap E \cap U| = 25 - 11 - 9 + 3 = 8$
     * **Only English:** $|E| - |H \cap E| - |E \cap U| + |H \cap E \cap U| = 26 - 11 - 8 + 3 = 10$
     * **Only Urdu:** $|U| - |H \cap U| - |E \cap U| + |H \cap E \cap U| = 26 - 9 - 8 + 3 = 12$
   * Total reading exactly one newspaper:
     $$\text{Exactly One} = 8 + 10 + 12 = \mathbf{30 \text{ people}}$$

3. **(iii) Number of people who read both Hindi and Urdu, but not English ($|(H \cap U) \setminus E|$):**
   * This corresponds to the intersection of Hindi and Urdu minus the intersection of all three:
     $$|(H \cap U) \setminus E| = |H \cap U| - |H \cap E \cap U| = 9 - 3 = \mathbf{6 \text{ people}}$$

(i) Define planar graph. Is $K_5$ a planar graph? Justify your answer.
(ii) Suppose that a connected planar graph has 30 edges. If a planar representation of this graph divides the plane into 20 regions, how many vertices does this graph have?

   * By **Euler's Formula** for connected planar graphs:
     $$V - E + R = 2$$
   * We are given $E = 30$ edges and $R = 20$ regions:
     $$V - 30 + 20 = 2 \implies V - 10 = 2 \implies V = \mathbf{12 \text{ vertices}}$$

### Question 4(b)
**Question:**
Let $a$ and $b$ be two numeric functions defined as follows:
$$a_r = \begin{cases} 2 & 0 \le r \le 2 \\ 2^{-r} + 5 & r \ge 3 \end{cases}$$
$$b_r = \begin{cases} 3 - 2^{-r} & 0 \le r \le 1 \\ r + 2 & r \ge 2 \end{cases}$$
Determine the sum and product of numeric functions $a$ and $b$.

---

#### Solution:

1. **Determine the Sum $s_r = a_r + b_r$:**
   * We partition the domain based on the interval boundaries ($r=1, r=2, r=3$):
     * **For $0 \le r \le 1$:**
       $$s_r = 2 + \left(3 - 2^{-r}\right) = 5 - 2^{-r}$$
     * **For $r = 2$:**
       $$a_2 = 2, \quad b_2 = 2 + 2 = 4 \implies s_2 = 2 + 4 = 6$$
     * **For $r \ge 3$:**
       $$s_r = \left(2^{-r} + 5\right) + (r + 2) = 2^{-r} + r + 7$$
   * **Sum function:**
     $$s_r = \begin{cases}
     5 - 2^{-r} & r = 0, 1 \\
     6 & r = 2 \\
     2^{-r} + r + 7 & r \ge 3
     \end{cases}$$

2. **Determine the Product $p_r = a_r b_r$:**
   * Evaluate the multiplication in the same intervals:
     * **For $0 \le r \le 1$:**
       $$p_r = 2 \times \left(3 - 2^{-r}\right) = 6 - 2^{1-r}$$
     * **For $r = 2$:**
       $$p_2 = a_2 \times b_2 = 2 \times 4 = 8$$
     * **For $r \ge 3$:**
       $$p_r = \left(2^{-r} + 5\right) \times (r + 2) = (r+2)2^{-r} + 5r + 10$$
   * **Product function:**
     $$p_r = \begin{cases}
     6 - 2^{1-r} & r = 0, 1 \\
     8 & r = 2 \\
     (r+2)2^{-r} + 5r + 10 & r \ge 3
     \end{cases}$$

### Question 5(a)
**Question:**
In how many ways can a committee of 5 members be formed from 11 Senators so that:
(i) Senator A is always included
(ii) Senator A is always excluded
(iii) At least one of Senator A and Senator B will be included

---

#### Solution:
We need to select combinations from 11 available senators:

1. **(i) Senator A is always included:**
   * A's seat on the committee is fixed.
   * We must choose the remaining $5 - 1 = 4$ committee members from the remaining $11 - 1 = 10$ senators:
     $$\text{Ways} = \binom{10}{4} = \frac{10 \times 9 \times 8 \times 7}{4 \times 3 \times 2 \times 1} = \mathbf{210 \text{ ways}}$$

2. **(ii) Senator A is always excluded:**
   * Senator A cannot be chosen.
   * We must choose all 5 committee members from the remaining $11 - 1 = 10$ senators:
     $$\text{Ways} = \binom{10}{5} = \frac{10 \times 9 \times 8 \times 7 \times 6}{5 \times 4 \times 3 \times 2 \times 1} = \mathbf{252 \text{ ways}}$$

3. **(iii) At least one of Senator A and Senator B is included:**
   * **Complementary Counting Method:**
     * Total ways to form the committee without restrictions:
       $$\text{Total} = \binom{11}{5} = \frac{11 \times 10 \times 9 \times 8 \times 7}{5 \times 4 \times 3 \times 2 \times 1} = 462 \text{ ways}$$
     * Ways where *both* Senator A and Senator B are excluded:
       $$\text{Both Excluded} = \binom{11 - 2}{5} = \binom{9}{5} = \frac{9 \times 8 \times 7 \times 6 \times 5}{5 \times 4 \times 3 \times 2 \times 1} = 126 \text{ ways}$$
     * Ways with at least one included:
       $$\text{At Least One} = \text{Total} - \text{Both Excluded} = 462 - 126 = \mathbf{336 \text{ ways}}$$

Determine whether the conclusion "We will be home by sunset" follows logically from the above set of premises.

#### Solution:
We will translate the premises into propositional logic and construct a formal proof:

1. **Define the propositional variables:**
   * $p$: "It is sunny this afternoon."
   * $q$: "It is colder than yesterday."
   * $r$: "We will go swimming."
   * $s$: "We will take a canoe trip."
   * $t$: "We will be home by sunset."

2. **Translate the premises and the conclusion:**
   * **Premise 1:** $\neg p \land q$ ("It is not sunny and it is colder than yesterday")
   * **Premise 2:** $r \rightarrow p$ ("We will go swimming only if it is sunny")
   * **Premise 3:** $\neg r \rightarrow s$ ("If we do not go swimming, we will take a canoe trip")
   * **Premise 4:** $s \rightarrow t$ ("If we take a canoe trip, we will be home by sunset")
   * **Conclusion:** $t$ ("We will be home by sunset")

3. **Construct the formal proof using rules of inference:**

| Step | Proposition | Justification |
| :--- | :--- | :--- |
| 1. | $\neg p \land q$ | Premise 1 |
| 2. | $\neg p$ | Simplification from (1) |
| 3. | $r \rightarrow p$ | Premise 2 |
| 4. | $\neg r$ | Modus Tollens from (2) and (3) |
| 5. | $\neg r \rightarrow s$ | Premise 3 |
| 6. | $s$ | Modus Ponens from (4) and (5) |
| 7. | $s \rightarrow t$ | Premise 4 |
| 8. | $t$ | Modus Ponens from (6) and (7) |

Since we successfully derived the conclusion $t$ using valid rules of inference, the conclusion **"We will be home by sunset" follows logically** from the premises.

What do you mean by graph invariant? Is the following pair of graphs isomorphic? Justify.

#### Solution:

1. **Graph Invariant Definition:**
   A graph invariant is a property of a graph that is preserved under isomorphism. If two graphs $G$ and $H$ are isomorphic, they must share the exact same values for all graph invariants (such as number of vertices, number of edges, degree sequence, connectivity, number of cycles of a specific length, chromatic number, etc.).

2. **Check Isomorphism of the given pair ($G_1$ and $G_2$):**
   Let's check the structural properties:
   * **Simple Invariants:**
     * Vertices ($V$): Both $G_1$ and $G_2$ have 6 vertices.
     * Edges ($E$): Both graphs have 9 edges.
     * Degree Sequence: Both graphs are 3-regular (each vertex has degree 3).
     * Bipartiteness: Neither graph is bipartite, as both contain 3-cycles (triangles).
       * In $G_1$: The triangles are $\{u_1, u_2, u_6\}$ and $\{u_3, u_4, u_5\}$.
       * In $G_2$: The triangles are $\{v_1, v_2, v_5\}$ and $\{v_3, v_4, v_6\}$.
   * **Construct a Bijective Isomorphism Mapping $f: V(G_1) \to V(G_2)$:**
     * Let's map the first triangle in $G_1$ to the first triangle in $G_2$:
       $$f(u_1) = v_5, \quad f(u_2) = v_2, \quad f(u_6) = v_1$$
     * Now use the matching edges connecting the two triangles in each graph to map the remaining vertices:
       * Since $(u_1, u_4)$ is an edge in $G_1$, and $(f(u_1), v_6) = (v_5, v_6)$ is an edge in $G_2$, map:
         $$f(u_4) = v_6$$
       * Since $(u_2, u_3)$ is an edge in $G_1$, and $(f(u_2), v_3) = (v_2, v_3)$ is an edge in $G_2$, map:
         $$f(u_3) = v_3$$
       * Since $(u_6, u_5)$ is an edge in $G_1$, and $(f(u_6), v_4) = (v_1, v_4)$ is an edge in $G_2$, map:
         $$f(u_5) = v_4$$
   * **Verify Adjacency Preservation:**
     Now we check if the remaining edges in the triangle $\{u_3, u_4, u_5\}$ of $G_1$ are preserved under $f$:
     * $(u_3, u_4) \in E(G_1) \implies (f(u_3), f(u_4)) = (v_3, v_6) \in E(G_2)$ (Valid!)
     * $(u_4, u_5) \in E(G_1) \implies (f(u_4), f(u_5)) = (v_6, v_4) \in E(G_2)$ (Valid!)
     * $(u_5, u_3) \in E(G_1) \implies (f(u_5), f(u_3)) = (v_4, v_3) \in E(G_2)$ (Valid!)

Since we have found a bijection $f$ that preserves all adjacency and non-adjacency relations, the two graphs are **isomorphic**.

   * $3^{16} \equiv 111^2 = 12321 \equiv \mathbf{66} \pmod{645} \quad (\text{since } 12321 = 19 \times 645 + 66)$

   * $3^{256} \equiv 396^2 \equiv (-249)^2 = 62001 \equiv \mathbf{81} \pmod{645} \quad (\text{since } 62001 = 96 \times 645 + 81)$

### Question 7(b)
**Question:**
Given a relation R on set A = {3, 5, 9, 15, 24, 45} such that:
$R = \{(a, b) : a \text{ is divisor of } b \text{ and } a, b \in A\}$
(i) Show that R is a POSET.
(ii) Draw Hasse Diagram.
(iii) Find the maximal and minimal elements in A.

---

#### Solution:

1. **(i) Show $R$ is a POSET:**
   * To show $R$ is a poset, we prove it is reflexive, antisymmetric, and transitive:
     * **Reflexive:** For any $x \in A$, $x$ divides $x$ ($x/x = 1$). Thus, $(x, x) \in R$ for all $x \in A$.
     * **Antisymmetric:** For any $x, y \in A$, if $(x, y) \in R$ and $(y, x) \in R$, then $x \mid y$ and $y \mid x$. Since elements of $A$ are positive integers, this implies $x = y$.
     * **Transitive:** For any $x, y, z \in A$, if $x \mid y$ and $y \mid z$, then $y = c_1 x$ and $z = c_2 y$ for some $c_1, c_2 \in \mathbb{Z}$. So $z = c_2 (c_1 x) = (c_1 c_2) x \implies x \mid z$. Thus, $(x, z) \in R$.
   * Since $R$ is reflexive, antisymmetric, and transitive, it is a partial order, and $(A, R)$ is a POSET.

2. **(ii) Draw the Hasse Diagram:**
   * Determine the divisibility relations:
     * $3 \mid 9, 15, 24, 45$
     * $5 \mid 15, 45$
     * $9 \mid 45$
     * $15 \mid 45$
   * Find immediate covers (transitive reductions):
     * $3$ is covered by: $9, 15, 24$
     * $5$ is covered by: $15$
     * $9$ is covered by: $45$
     * $15$ is covered by: $45$
   * **Layout:**
     * **Level 3 (Top):** `24` and `45`.
     * **Level 2 (Middle):** `9` and `15`.
       * Draw edges from `3` to `9`, `3` to `15`, and `3` to `24`.
       * Draw edge from `5` to `15`.
       * Draw edges from `9` to `45`, and `15` to `45`.
     * **Level 1 (Bottom):** `3` and `5`.

```
       [24]       [45]
         |        /  \
         |       /    \
         |     [9]    [15]
         |      |     /  \
         |      |    /    \
         |      |   /      \
       [ 3 ]-------/       [ 5 ]
```

3. **(iii) Maximal and Minimal Elements:**
   * **Maximal elements** (elements that divide no other element in $A$): **$\{24, 45\}$**
   * **Minimal elements** (elements that are divided by no other element in $A$): **$\{3, 5\}$**
